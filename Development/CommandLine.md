### linux 命令
#### grep 和 find 忽略大小写

`grep` 使用 -i 参数来忽略大小写
```bash
grep -i 'xxxXXX' filePath
```

`find` 使用 -iname参数来忽略大小写：

```bash
find filesPath -iname 'xxxXXX' 
```
